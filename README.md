# File_Manager
A linux file manager made in C++ and GTK+

v0.2 Features:
1. Manage Files(Copy, Cut, Paste, Delete, Rename)
2. File Editor
3. Image Viewer supports jpg, jpeg, png and gif
4. Video Player (mp4)

Note: Its an unfinished project, more features will be added in 0.3
